Prediction and Response System

Igor Kapusniak | 20102236

Applied Computing: Internet of Things

Real-Time Trajectory Prediction and Response Using a Quadruped Robot

This project explores the concept of a real-time object detection and response system built on the Unitree Go2 quadruped robot. The system uses the robot's onboard camera and LiDAR sensor to detect moving objects, predict their trajectory, and execute an appropriate physical response. Two demonstration scenarios are targeted: keeping a balloon airborne by intercepting it before it hits the ground, and evading fast-moving hazards such as a falling object. The project explores the intersection of computer vision, trajectory prediction, and motion control on an affordable and accessible robotics platform.
